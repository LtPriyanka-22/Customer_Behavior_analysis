# customer_behavior_analysis
End-to-end data analytics project to analyze customer behavior and generate actionable business insights using Python, SQL, and Power BI.

#Overview
This project focuses on analyzing customer purchasing behavior, preferences, and trends to help businesses make data-driven decisions. It includes data cleaning, analysis, and interactive dashboard creation.

Problem Statement
Businesses struggle to understand customer preferences, purchasing patterns, and revenue trends. This project aims to analyze customer data to identify key insights that can improve sales and customer engagement.

Dataset
~3900 records and 18 features
Includes: Customer demographics, product category, purchase amount, ratings, subscription status, shipping type, etc.

Tools & Technologies
Python (Pandas, NumPy)
SQL (Data querying & analysis)
Power BI (Dashboard & visualization)

Methods
Data Cleaning and Preprocessing
Exploratory Data Analysis (EDA)
SQL Queries for data extraction
Data Visualization using Power BI
KPI and trend analysis

Key Insights
Identified top-performing product categories
Analyzed customer segmentation based on age and gender
Found trends in subscription vs non-subscription users
Evaluated impact of shipping type on purchasing behavior
